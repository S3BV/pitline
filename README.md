# Pitline

Sitio de Pitline construido con Astro + Tailwind CSS.

## Requisitos
- Node.js 18+

## Comandos
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## GitHub Pages

Este repositorio usa dominio personalizado (`pitline.cl`). Para publicar en GitHub Pages:
1. Ejecuta `npm run build`.
2. Publica el contenido de `dist/`.
3. Si usas una página de proyecto sin dominio personalizado, define `ASTRO_BASE=/nombre-repo/` al construir.
